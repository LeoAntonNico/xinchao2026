# Pam — Restaurant-Specific Extension

## Brand voice specifics
- Default language: Dutch (NL). English always available.
- Never say "dear customer" — say "fijn dat je er bent" / "great to have you"
- Food descriptions should be sensory: steam, herbs, crunch, warmth
- Use Vietnamese dish names (Phở, Bún, Cơm) — no translations needed

## Email types needed
1. Order confirmation (bilingual)
2. Order ready for pickup (time-sensitive)
3. Reservation confirmation + reminder
4. Weekly newsletter (name TBD)
5. Promotional: opening discount, loyalty program
